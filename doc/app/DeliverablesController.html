Passes all know deliverables types and complexities to Deliverable model to get minimum, average, and maximum of effort, size and rate. Input params: DeliverableType, and Complexity Returns : Hash of min, avg, and max of each: effort, size, and rate 

     # File app/controllers/deliverables_controller.rb, line 105
105:    def dynamic_estimates
106:     estimates = []
107:     deliverable_types = DeliverableType.find(:all)
108:     for deliverable_type in deliverable_types
109:       for complexity in Complexity.getValues
110:         estimates << {:type => deliverable_type.name, :complexity => complexity,

113:     end
